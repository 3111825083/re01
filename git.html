1. "git add . 加 git commit "将本地修改好的代码保存并同步到git中
  2. "push -u origin main"将git同步到上游
  3. "clone 网址 ."克隆云仓库到本地
  4. 当使用pull or push时遇到：fatal: refusing to merge unrelated
  5. histories在其后添加--allow-unrelated-histories即可解决
  6. "remote add upstream 网址"添加上游代码库
  7. "remote -v"查看仓库链接
  8. "git checkout -b name"创建分支
